UPI Redirection Example

This API allows you to generate a UPI payment link by specifying the following details:

  • VPA (Virtual Payment Address): Your UPI ID (e.g., yourname@bank)
  • Bank: The bank name associated with your UPI ID (e.g., icici)
  • Amount: The amount to be paid (e.g., 100)

To use the API, you can either input the details in the form below or manually construct the URL in this format:

upi-redirection-api.vercel.app/upi/vpa@bank&amount
